# GameHunt

**GameHunt** - браузерная игра-головоломка, в которой нужно угадать загаданную видеоигру по подсказкам. Вдохновлена Wordle и Contexto, но адаптирована под тематику видеоигр.

## Как играть

1. Введи название любой видеоигры в поле ввода
2. Система сравнит ее с загаданной и подсветит подсказки цветом:
   * **Серый** - нет совпадений
   * **Оранжевый** - частичное совпадение
   * **Зеленый** - полное совпадение
3. Анализируй совпадения и угадай игру!

### Логика сравнения
| Подсказка | Оранжевый | Зеленый |
|---|---|---|
| Год выпуска | Разница <= 3 лет | Точное совпадение |
| Жанры | Хотя бы один общий | Все совпадают |
| Платформы | Хотя бы одна общая | Все совпадают |
| Кол-во продаж | Разница <= 25% | Точное совпадение |

## Уровни сложности

| Уровень | Описание |
|---|---|
| Легкая | Все 4 подсказки открыты |
| Средняя | Перед игрой нужно выбрать 2 подсказки из 4. Остальные будут скрыты |
| Сложная | Все подсказки скрыты |

## Возможности

- **Автодополнение** - при вводе отображаются подходящие варианты из БД, поиск работает и по псевдонимам (например, "гта 5" -> "GTA V")
- **Дополнительная подсказка** - после 8 неудачных попыток появляется кнопка для открытия зашифрованного названия игры
- **Статистика** - по окончании игры показывается загаданная игра, количество попыток и уровень сложности
- **Сохранение прогресса** - прогресс автоматически сохраняется в localStorage без регистрации
- **Конфетти** - анимация при победе
- **Адаптивный интерфейс** - работает на мобильных устройствах

## База данных

Более 150 игр различных жанров и платформ, включая мобильные. Каждая игра содержит:
- Название и псевдонимы
- Год выпуска
- Жанры (3 шт)
- Платформы (2-3 шт)
- Количество продаж / скачиваний

## Технический стек

| Технология | Назначение |
|---|---|
| React + Vite | Основной фреймворк и сборщик |
| Tailwind CSS | Стилизация |
| Supabase (PostgreSQL) | Облачная база данных |
| supabase-js | Клиент для работы с БД |
| canvas-confetti | Анимация конфетти |
| vite-plugin-svgr | Поддержка SVG как React компонентов |

## Запуск проекта

```
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка
npm run build
```

### Переменные окружения

Создайте файл .env в корне проекта с такими ключами:

```
VITE_SUPABASE_URL=ваш_supabase_url
VITE_SUPABASE_PUBLISHABLE_KEY=ваш_anon_key
```
